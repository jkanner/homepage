HTML Guide

HTML is used by web browsers to display pages. It uses "tags", where tags tell the browser how to format the text. Here are a few tags you are using in your web page:

	    h2  -  Heading 2 - A smaller heading

	    a   -  Link      - Link to another page

	    img - image      - Display an image
